# Claude Skills Library

Personal collection of Claude Code skills for business automation, marketing, and design.

## Structure

```
.claude/skills/
├── business/
│   ├── ads-audit/      # Ad campaign analysis
│   ├── crm-hub/        # Lead generation & CRM
│   └── marketing/      # SEO, email, CRO, content
├── design/
│   └── ui-ux-pro-max/  # UI/UX, branding, design systems
└── tools/
    ├── website-cloner/ # Analyze & replicate sites
    └── explain.md      # Code explanation helper
```

## Usage

Clone this repo. Skills become available in Claude Code via `/skill-name`.
