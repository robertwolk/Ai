# Explain Code

When the user runs /explain, analyze the selected code or file and provide:

1. **Purpose** — What the code does in plain English
2. **How it works** — Step-by-step walkthrough of the logic
3. **Key concepts** — Any AI/ML concepts, algorithms, or patterns used
4. **Suggestions** — Potential improvements or things to watch out for

Keep explanations beginner-friendly since this is a learning repository.
